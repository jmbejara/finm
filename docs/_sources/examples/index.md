## Overview

### Fixed Income

- **[Replicate GSW Yield Curve](01_replicate_GSW_yield_curve_ipynb.ipynb)**: Reproduce the Gurkaynak, Sack, and Wright (2006) Treasury yield curve methodology using the Nelson-Siegel-Svensson model.

### Corporate Bonds

- **[Corporate Bond Returns](02_corp_bond_returns_ipynb.ipynb)**: Introduction to pulling and analyzing corporate bond data from WRDS.
- **[Corporate Bond Returns - HKM Methodology](03_corp_bond_returns_ftsfr_HKM_ipynb.ipynb)**: Data cleaning procedures based on He, Kelly, and Manela methodology following Nozawa (2017).

### Factor Models

- **[Factor Regression Example](05_factor_regression_example_ipynb.ipynb)**: Demonstrates CAPM and Fama-French 3-factor model regressions, including Jensen's alpha and factor beta interpretation.

### Data Integration

- **[Data Integration Tests](04_data_integration_tests_ipynb.ipynb)**: Tests and demonstrates integration of multiple data sources including Fama-French factors, Federal Reserve data, HKM data, and Open Source Bond data.
