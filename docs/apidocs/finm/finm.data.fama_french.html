finm.data.fama_french#

Fama-French factor data module.

Provides access to Fama-French 3 factors from Ken French’s Data Library.

Website: https://mba.tuck.dartmouth.edu/pages/faculty/ken.french/data_library.html

Standard interface: - pull(data_dir, accept_license): Download data from source - load(data_dir, format): Load cached or bundled data (returns polars) - to_long_format(df): Convert to long format

License: Copyright Fama & French. Please cite the papers when using this data.

finm.data.fama_french.pull(data_dir: pathlib.Path | str, start: str | datetime.datetime | None = None, end: str | datetime.datetime | None = None, frequency: finm.data.fama_french.FrequencyType = 'daily', accept_license: bool = False) → pandas.DataFrame[source]#

Download Fama-French factors from Ken French’s Data Library.

Requires pandas_datareader to be installed.

Website: https://mba.tuck.dartmouth.edu/pages/faculty/ken.french/data_library.html

Parameters

data_dir : Path or str Directory to save downloaded data. start : str or datetime, optional Start date of the data. Format: ‘YYYY-MM-DD’. end : str or datetime, optional End date of the data. Format: ‘YYYY-MM-DD’. frequency : {“daily”, “monthly”}, default “daily” Data frequency. accept_license : bool, default False Must be set to True to acknowledge the data provider’s terms.

Returns

pd.DataFrame DataFrame containing the factors (as decimals): - Mkt-RF: Excess return on the market - SMB: Small Minus Big (size factor) - HML: High Minus Low (value factor) - RF: Risk-free rate

Raises

ValueError If accept_license is False.
